# Test Videoları

Bu klasöre kaydedilen demo videoları buraya koyun.

## test_hard_course_5ep_8fps.mp4 nasıl oluşturulur?

1. Script'i çalıştırın:
   ```bash
   python test_hard_course_pygame.py --model models_hybrid_2/best/best_model --n_episodes 5 --fps 8
   ```

2. Açılan Pygame penceresini ekran kaydı ile kaydedin:
   - **Windows:** `Win + G` → Oyun çubuğu → Kayıt
   - **OBS Studio:** Ekran kaydı

3. Videoyu `test_hard_course_5ep_8fps.mp4` adıyla bu klasöre kaydedin.

4. README.md'deki video linki bu dosya adına göre ayarlanmıştır.
